Datenschutzerklärung

Letzte Aktualisierung: Januar 2026

ZenMaster Sudoku respektiert Ihre Privatsphäre. Diese Erklärung beschreibt den Umgang mit Daten innerhalb der App.

Datenerhebung

ZenMaster Sudoku erhebt keine personenbezogenen Daten, die Nutzer direkt identifizieren.

  • Ein Spielername kann lokal auf dem Gerät gespeichert werden.
  • Dieser Name wird nicht an Server übertragen.
  • Es wird kein Benutzerkonto erstellt.

Werbung

Die App kann Werbung von Drittanbietern anzeigen.

  • Drittanbieter können anonymisierte technische Daten erfassen.
  • ZenMaster Sudoku hat keinen Zugriff auf diese Daten.

Kinder

ZenMaster Sudoku ist für alle Altersgruppen geeignet.

Sicherheit

Alle lokalen Daten bleiben ausschließlich auf dem Gerät des Nutzers.
